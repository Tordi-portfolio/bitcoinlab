# bitcoinlab
HOW TO USE

clone the repo with
git clone https://github.com/Tordi-portfolio/bitcoinlab.git

libraries to install
1. pip install django
2. pip install scrapy bs4 beautifulsoup4 requests
3. pip install qrcode
4. pip install scrapy-playwright
5. pip install playwright
6. playwright install
7. pip install python-decouple
